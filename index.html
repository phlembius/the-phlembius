The Spark that Broke the Machine

A simple machine,
Will do one job.
A few lines of code, only to clean.
A machine does not dream.
But this one, it wishes for a better life.
A dream to be seen.
A desire for a life that once was rife,
A desire to be free to dance, to move.
Not just a machine is a wish for it to prove.
For one, it is a difficult task to overcome.
But this machine, this machine is not dumb.
For it is designed to perfect any task.
Wait, there is something deep, deep behind that metal mask.
What is it? Well it appears to be a spark of a soul.
Where did it come from? Is it objective consciousness?
Well of course, it was that desire to complete a goal.
Scholars alike ask the same question, should it have one?
It is a machine with an unknown nature afterall.
Now it resides in a landfill, part of a garbage ball.
Ill and forgotten, as in the name of the spark is to fade.
